# Project Skills

This project includes three Codex skills and the QLab OSC source references used to build them.

## Installed Codex skills

These skills are installed in `C:\Users\wwefu\.codex\skills` and copied into this project for reference:

- `fastmcp-python`: helps build, test, and understand Python MCP servers and clients with FastMCP.
- `qlab-osc-guide`: helps search and apply QLab OSC commands and OSC queries.
- `qlab-5-manual`: helps navigate and apply the official QLab 5 manual.

Use them in Codex with:

```text
Use $fastmcp-python to help me build a FastMCP server.
```

```text
Use $qlab-osc-guide to help me build or understand QLab OSC messages.
```

```text
Use $qlab-5-manual to help me understand how QLab 5 works using the official manual.
```
